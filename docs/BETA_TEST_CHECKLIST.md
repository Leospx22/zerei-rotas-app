## 2. Test Flow

- [ ] Criar conta
- [ ] Completar Perfil
- [ ] Importar rota real
- [ ] Revisar Rota
- [ ] Confirmar que paradas sem numero aparecem como `#P`
- [ ] Conferir avisos de endereco duplicado com os numeros das paradas correspondentes
- [ ] Mostrar no mapa
- [ ] Confirmar que enderecos duplicados compartilham coordenadas quando uma parada equivalente ja tem coordenada valida
- [ ] Testar "Copiar endereco" quando aparecer "Insira o endereco manualmente"
- [ ] Iniciar entrega
- [ ] Navegar para uma parada
- [ ] Entregar um pacote e fechar o app completamente
- [ ] Reabrir o app e confirmar "Continuar entrega"
- [ ] Confirmar que a rota mostra "Rota restaurada. Voce pode continuar a entrega." apenas uma vez
- [ ] Colocar o telefone sem internet e continuar uma acao local da rota
- [ ] Reabrir o app sem internet e confirmar que a rota continua salva no aparelho
- [ ] Registrar ocorrencia
- [ ] Continuar rota
- [ ] Ver Historico

## 4. Known Beta Limitations

- Pagamento ainda nao esta ativo.
- O app esta em evolucao e pode mudar durante o teste.
- Algumas coordenadas podem depender da qualidade dos dados da planilha.
- Se uma parada mostrar "Insira o endereco manualmente", use "Copiar endereco" ou "Navegar" para buscar manualmente no GPS.
- `#P` significa que a planilha nao trouxe um numero de Stop valido para aquela parada.
- O progresso da rota fica salvo localmente neste aparelho; ainda nao existe sincronizacao em nuvem entre celulares.
- Se a internet cair, a rota local deve continuar funcionando. Perfil, funil e outras acoes Supabase podem esperar a conexao voltar.
- O teste inicial e focado em Android.
